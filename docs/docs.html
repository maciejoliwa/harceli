Getting Started

Installation

First off, launch your terminal and put in:

npm i -g harceli

Wait for the library to install.

Usage

Now, in your project's directory create harceli.config.json

Fill it with this content:

{
"watched": [
{ "files": ["test1.js", "test2.js"], "on_write": "echo files changed!" },
{ "files": "app.py", "on_write": "echo python file changed!" }
]
}

Take note, that the files property can point either to an array of strings, or to a string.

Run the harceli command and make changes to the files in your directory. You should see on_write commands getting called everytime you change and save the file!

Of course, there are many things you can do using this system. For example you could execute a python/node.js script everytime you change one of your files. You can automatically add them to git and many more!
